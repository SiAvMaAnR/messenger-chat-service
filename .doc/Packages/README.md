### Package: list

`dotnet list package`

### Package: add

`dotnet add package <PACKAGE_NAME>`

`dotnet add package <PACKAGE_NAME> -v <VERSION>`

### Package: remove

`dotnet remove package <PACKAGE_NAME>`

### Package: update-list

`dotnet outdated`

### Package: update-all

`dotnet outdated -u`

### Package: restore

`dotnet restore`
